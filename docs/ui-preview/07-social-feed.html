Chia sẻ custom hook useDebounce mà mình hay dùng trong dự án. Rất hữu ích khi làm tính năng search realtime, tránh gọi API liên tục khi user đang gõ 💡

#react #customhook #typescript

TypeScript 
const useDebounce = <T>(value: T, delay: number): T => {
  const [debouncedValue, setDebouncedValue] = useState<T>(value);

  useEffect(() => {
    const timer = setTimeout(() => {
      setDebouncedValue(value);
    }, delay);

    return () => clearTimeout(timer);
  }, [value, delay]);

  return debouncedValue;
};
