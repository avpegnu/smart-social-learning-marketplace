useEffect khác gì useLayoutEffect?
Khác biệt chính:
- useEffect chạy sau khi trình duyệt paint, không block UI.
- useLayoutEffect chạy trước paint, block UI cho tới khi xong.

Dùng useLayoutEffect khi cần đo DOM hoặc tránh flicker.
