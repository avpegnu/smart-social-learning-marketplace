TN
Cleanup function trong useEffect sẽ chạy khi component unmount hoặc trước khi effect chạy lại. Nó rất hữu ích để dọn dẹp subscriptions, timers, etc. Ví dụ: 
useEffect(() => {
  const timer = setInterval(() => {
    console.log('tick');
  }, 1000);

  // Cleanup function
  return () => {
    clearInterval(timer);
  };
}, []);
Khi component unmount, clearInterval sẽ được gọi để tránh memory leak. Bạn hiểu chưa?
