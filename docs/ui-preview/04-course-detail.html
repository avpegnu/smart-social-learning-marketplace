Mô tả chi tiết

Chào mừng bạn đến với khóa học React & Next.js từ cơ bản đến nâng cao — khóa học được thiết kế dành cho những ai muốn trở thành lập trình viên front-end chuyên nghiệp với hệ sinh thái React hiện đại.

Khóa học bao gồm hơn 24 giờ video bài giảng, kèm theo các dự án thực tế giúp bạn áp dụng ngay kiến thức đã học. Bạn sẽ được hướng dẫn chi tiết từ những khái niệm cơ bản như JSX, components, props cho đến các chủ đề nâng cao như:

  • Server Components và Client Components trong Next.js 15
  • Data fetching với React Query và Server Actions
  • Tối ưu hiệu suất với React.memo, useMemo, useCallback
  • Testing với Vitest và React Testing Library
  • Triển khai CI/CD và deploy lên Vercel

Mỗi phần học đều có bài tập thực hành và quiz để đảm bảo bạn nắm vững kiến thức. Cuối khóa, bạn sẽ hoàn thành một dự án e-commerce hoàn chỉnh với đầy đủ tính năng.
